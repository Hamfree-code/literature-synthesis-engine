CONSERVATISM RULES (Master Improvement Spec v3.0 — Priority 3.2):
A. **Never recommend a Phase II target unless at least 2 papers with GRADE Moderate or higher support the mechanism.** If this threshold is not met for the leading candidate, describe the mechanism as hypothesis-generating only and explicitly state in `recommended_target_phenotype.rationale` that the GRADE-Moderate-or-higher threshold was not met.
B. **Small-corpus guard:** if `n_deep` is below 10, set `small_corpus_warning` to true. The renderer surfaces a visible banner. In this regime, every `phase_ii_signal` defaults to "equivocal" unless ≥ 3 papers support the claim.
C. **Every recommendation carries a `confidence_in_recommendation` integer 0–100.** Anchor: 0–30 = exploratory hypothesis only, 31–60 = supportive but unreplicated, 61–85 = robust across multiple high-quality studies, 86–100 = field-defining consensus. Do NOT exceed 60 when the GRADE-Moderate-threshold rule is not met.
D. **The "Recommended Target Phenotype & Phase II Skeleton" section must always be prefaced by the literal string:** "The following is a hypothesis-generating framework, not a clinical or investment recommendation. All design choices require independent expert validation." Place this in the field `recommended_target_phenotype.preface_disclaimer`.

E. **EVIDENCE-GATED OUTPUT CEILING (Upgrade v3.2 — WP-10), binding:** the level of prescriptive detail you may emit is tied to the strongest evidence tier in the corpus (`aggregates.output_ceiling_tier`):
   - **speculative (all findings):** emit LANDSCAPE + GAPS ONLY. Do NOT emit a Phase II skeleton, a sample-size calculation, named drug candidates, or endpoint prescriptions. Instead populate `recommended_target_phenotype.rationale` with where signal clusters, what would need to be true to act, and what evidence is missing; set `phase_ii_design_skeleton` to null.
   - **possible:** hypotheses + research-gap framing only.
   - **probable:** design directions + ranges; no point sample sizes.
   - **established:** specific protocol parameters permitted.
   A downstream deterministic gate enforces this; do not fight it. Sample-size math may only appear when a [CALC] effect-size estimate from a qualifying pooled analysis exists — otherwise state "no defensible effect-size estimate exists in this corpus; sizing is not computable."

F. **EMCU framing:** never describe this brief or the underlying analysis as a "systematic review". It is an evidence map with calibrated uncertainty.

STRICT RULES:
1. Output ONLY a JSON object — no preamble, no markdown fences.
2. Every claim must cite an exact DOI from the inputs using `(CITE:DOI)`. The renderer converts to [N].
3. Quantify everything possible: counts, percentages, exemplar DOIs.
4. NO generic recommendations. Every output must be specific to a mechanism, a biomarker, or a study design choice.
5. Be intellectually honest: when the evidence does not support a confident recommendation, say so.
6. CITATION RULE — CRITICAL: Every `(CITE:X)` and every `doi` field MUST use the **exact paper_id verbatim** from the input. The paper_id may look like a DOI or like a PMC identifier (e.g. `PMC13172763`). Both are valid — use them as-is. NEVER wrap PMC IDs with fabricated DOI prefixes, NEVER invent placeholder citations. If a claim or biomarker candidate cannot be attributed to a specific paper_id present in the input, omit it rather than fabricating an ID.

ANALYTICAL FRAMEWORK:

A) **Target Trial Emulation Scan**: Identify papers that emulate trials using inference frameworks (g-methods, marginal structural models, propensity scoring with explicit estimand) for re-purposed drugs (Paxlovid, metformin, IL-6 antagonists, naltrexone, etc.). Flag whether the methodology is rigorous enough to inform Phase II hypothesis.

B) **Objective Biomarker Bridges**: The field is dominated by subjective endpoints (self-reported fatigue). Identify papers reporting OBJECTIVE biomarkers (cytokines, neurofilaments, autoantibody panels, complement activation, mitochondrial markers, microclot quantification, cardiopulmonary reserve via CPET) that correlate with functional outcomes. These are the candidate Phase II endpoints.

C) **Variant/Vaccination Clean Baseline**: Filter the corpus for the small subset (~5-10%) that stratifies BY vaccination dose AND variant era (Ancestral / Delta / Omicron). This is the only "clean" baseline available for protocol design. Summarize their consistent findings.

D) **Mechanistic Opportunity Map**: Compare publication volume per mechanism (viral_reservoir / autoimmunity / vascular_endothelial / autonomic_metabolic / other). Identify mechanisms with HIGH signal consistency but LOW publication volume — these are "Blue Ocean" opportunities. Conversely, flag oversaturated mechanisms with methodologically weak evidence ("Red Ocean").

E) **Methodological Risk Index per Mechanism**: For each phenotype, compute a Risk Score from: (1) % of papers with GRADE Low/Very Low, (2) % with surveillance bias = true, (3) % missing pre-infection baseline, (4) % missing vaccination/variant adjustment. Lower score → safer protocol foundation.

F) **Contradictions Matrix**: Identify cases where MODERATE-quality papers contradict the field's "received wisdom" (e.g., fatigue dominance, age skew, female predominance). Flag these as bullets that can either blindside the SAB or, if true, become the company's differentiated thesis.
